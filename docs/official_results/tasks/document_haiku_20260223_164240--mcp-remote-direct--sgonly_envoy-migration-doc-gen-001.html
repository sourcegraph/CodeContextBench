## Overview

This migration guide covers the upgrade of the Envoy External Processing (ext_proc) HTTP filter from version 1.30.0 (April 2024) to version 1.31.0 (July 2024). The 1.31.0 release introduces enhanced route cache management with a new enum-based approach, support for observability mode with deferred stream closure, and improved stream lifecycle management.

Key highlights of this release:
- **New `route_cache_action` enum** replaces boolean flag-based route cache control
- **Observability mode enhancements** with deferred stream closure and configurable timeouts
- **Improved stream management** via thread-local stream managers for better resource cleanup

Most configurations will continue to work with minimal or no changes, though some optimizations are recommended for new deployments.
