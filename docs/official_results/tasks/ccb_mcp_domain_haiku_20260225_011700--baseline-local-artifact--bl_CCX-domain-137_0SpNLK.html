   - Orchestrates the entire measure-layout-draw cycle through `performTraversals()`

3. **Choreographer.java** (`core/java/android/view/Choreographer.java`) - VSYNC timing coordinator that:
   - Synchronizes rendering with display refresh via `doFrame()`
   - Schedules frame work with `scheduleFrameLocked()`

4. **ThreadedRenderer.java** (`core/java/android/view/ThreadedRenderer.java`) - Hardware-accelerated rendering engine that:
   - Intercepts `draw()` calls and routes them to GPU
   - Manages off-thread rendering

5. **RecordingCanvas.java** (`graphics/java/android/graphics/RecordingCanvas.java`) - Display list recorder that:
   - Records draw operations for deferred GPU replay
   - Enables efficient hardware-accelerated rendering

**Rendering Flow:**
`Choreographer.doFrame()` → `ViewRootImpl.performTraversals()` → `View.measure()` → `View.layout()` → `View.draw(RecordingCanvas)` → `ThreadedRenderer.draw()`
