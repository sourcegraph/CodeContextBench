## Summary
This implementation adds support for HTTP/3 stream priority configuration to curl, implementing RFC 9218 Extensible Priorities with urgency (0-7) and incremental (boolean) parameters.

## Changes Made

### 1. Core Library Changes

#### `include/curl/curl.h` (Line ~2215)
- **Added**: New curl option `CURLOPT_H3_STREAM_PRIORITY` (ID: 324, type: CURLOPTTYPE_STRINGPOINT)
- Format: String in the format "u=N[,i]" where N is urgency (0-7) and i is optional incremental flag
- Example: "u=3" or "u=2,i"

#### `lib/urldata.h` (Curl_data_priority struct)
- **Added**: Two new fields to `struct Curl_data_priority`:
  - `unsigned char urgency;` - Stream urgency value (0-7, default 3 per RFC 9218)
  - `BIT(incremental);` - Incremental dependency flag
- Fields are guarded with `#ifdef USE_HTTP3` for HTTP/3-specific functionality
- Updated comment to remove TODO about RFC 9218 and document both HTTP/2 and HTTP/3 support
