   - `ShareCoordinatorService`, `GroupCoordinatorService` - manage coordinator state
   - `TransactionCoordinator` - exemplifies the reverse-order shutdown pattern

### Key Pattern Characteristic:

- **Startup order**: scheduler → txnManager → txnMarkerChannelManager
