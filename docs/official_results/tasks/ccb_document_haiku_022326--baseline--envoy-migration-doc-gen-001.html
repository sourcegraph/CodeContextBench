# Envoy External Processing Filter Migration Guide
## Version 1.30.0 → 1.31.0

## Overview

This guide documents the migration path for upgrading the Envoy External Processing (ext_proc) HTTP filter from version 1.30.0 (April 2024) to version 1.31.0 (July 2024).

The v1.31.0 release introduces significant improvements to observability mode and route cache management, while maintaining backward compatibility for most existing configurations. However, several important changes require attention:

- **1 Breaking Change**: A field type change in the external processor response protocol
- **1 Deprecated Field**: `disable_clear_route_cache` is superseded by a new enum-based `route_cache_action` field
- **3 New Configuration Options**: Enhanced route cache control and stream lifecycle management
- **Enhanced Observability Mode**: Improved support for non-blocking monitoring of traffic

The release focuses on improving observability mode with better stream lifecycle management and providing more granular control over route cache behavior.
