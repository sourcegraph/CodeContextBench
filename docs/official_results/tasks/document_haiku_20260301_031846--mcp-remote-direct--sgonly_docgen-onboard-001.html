## Build Prerequisites

### Required Go Version

Istio requires **Go 1.25.0 or later**. Verify your installation:

```bash
go version
```

### Environment Setup

Istio uses a modular build system that can operate in two modes:

#### Option A: Native Build (Recommended for Development)

If you have Go toolchain installed locally, you can build directly:

```bash
# No additional setup needed - the Makefile will configure itself
make build
```

#### Option B: Container-Based Build

To use Docker for a reproducible build environment:

```bash
export BUILD_WITH_CONTAINER=1
make build
```

**Note:** The build system will automatically detect your environment and set up necessary variables via `common/scripts/setup_env.sh`.

### Cloning the Repository with Submodules

Clone Istio with all submodules:

```bash
git clone --recurse-submodules https://github.com/istio/istio.git
cd istio
```

If you've already cloned without submodules, initialize them:

```bash
git submodule update --init --recursive
```
