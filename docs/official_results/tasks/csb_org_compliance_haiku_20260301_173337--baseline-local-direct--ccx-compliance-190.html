   - Verifies cryptographic signatures using OpenPGP

2. **Signature Validation on Load** (`pkg/plugins/manager/signature/signature.go`)
   - Validates plugin signatures during the validation pipeline stage
   - Enforces signature policies (rejects invalid/modified signatures)
   - Handles parent-child plugin signature inheritance

   - Allows dev mode to bypass restrictions
   - Blocks unsigned plugins not in the allowlist

4. **Key Management** (`pkg/plugins/manager/signature/statickey/static_retriever.go`)
   - Provides embedded PGP public key for signature verification

### Integration Points:
- **Bootstrap stage** calculates signatures when constructing plugins
- **Validation stage** enforces signature policies during load
- **Configuration** defines the unsigned plugins allowlist
- **Models** define signature statuses, types, and data structures
