- Distributed tracer initialization with instrumentation scopes
- Retrieves tracers via: `otel.GetTracerProvider().Tracer("scope-name")`
- Creates spans with inline attributes: `tracer.Start(ctx, "name", trace.WithAttributes(...))`
- Provides helper utilities like `TracingStart()` for conditional span creation
- Sets status with both error and success cases: `span.SetStatus(codes.Error/Ok, message)`

**Common Patterns:**
- Attribute setting: `span.SetAttributes(attribute.String/Int/Float64(...))`
- Event recording: `span.AddEvent("name", trace.WithAttributes(...))`
- Span context retrieval: `trace.SpanFromContext(ctx)`
- Proper cleanup: `defer span.End()`
