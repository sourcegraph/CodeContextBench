I found all 9 main scheduler framework plugin extension point interfaces defined in `staging/src/k8s.io/kube-scheduler/framework/interface.go`:

1. **PreFilterPlugin** - Called at the beginning of the scheduling cycle
2. **FilterPlugin** - Filters out nodes that cannot run a pod
3. **PostFilterPlugin** - Called after a pod cannot be scheduled

9. **PostBindPlugin** - Called after a pod is successfully bound
