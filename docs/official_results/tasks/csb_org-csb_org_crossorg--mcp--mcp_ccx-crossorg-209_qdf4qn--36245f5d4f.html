### Key Findings:

**Kubernetes** uses a sophisticated **pluggable health checker framework**:

**Grafana** uses a **state-based approach with database checks**:

- Uses `HealthNotifier` for thread-safe readiness signaling
- `/api/health` endpoint checks database connectivity and returns JSON with database status and version info
- Simpler middleware-based registration

**Prometheus** uses **state-based readiness with graceful shutdown tracking**:
- Endpoints: `/-/healthy`, `/-/ready`
- Three-state system: Ready, NotReady, Stopping
- `testReady()` middleware wraps handlers to enforce readiness checks
- Includes `X-Prometheus-Stopping` header to indicate graceful shutdown state
- Both endpoints support GET and HEAD methods
