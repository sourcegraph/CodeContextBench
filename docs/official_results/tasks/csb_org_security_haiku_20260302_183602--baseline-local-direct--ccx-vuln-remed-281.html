### **Internal ChildProcess Class** (lib/internal/child_process.js)

- **ChildProcess.prototype.spawn()** - Method that delegates to C++ binding

- **ProcessWrap** class (src/process_wrap.cc) - Wraps libuv process handles
  - **ProcessWrap::Spawn()** - Native spawn implementation
  - **ProcessWrap::Initialize()** - Binds methods to JavaScript
  - **ProcessWrap::Kill()** - Sends signals to child processes

- **SyncProcessRunner** class (src/spawn_sync.h) - Handles synchronous spawning

The null-byte validation prevents command injection by:
1. Validating file paths, arguments, environment variables, and shell paths for null bytes
2. Using `StringPrototypeIncludes(arg, '\u0000')` to detect embedded null bytes
3. Throwing `ERR_INVALID_ARG_VALUE` errors when detected
4. Applying validation consistently across all entry points before reaching C++ code
