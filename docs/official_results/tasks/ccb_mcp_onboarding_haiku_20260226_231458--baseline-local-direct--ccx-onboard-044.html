## Summary of Key Type Inference Files

**1. InferCtxt** (`compiler/rustc_infer/src/infer/mod.rs`, line 231)
- The main inference context struct that serves as the central hub for type inference
- Contains `InferCtxtInner` which holds type variable storage, unification tables, and region constraints
- Manages all type inference operations in the compiler

**2. FnCtxt** (`compiler/rustc_hir_typeck/src/fn_ctxt/mod.rs`, line 44)
- The function-level type checker for checking function bodies, closures, and const expressions
- Wraps `InferCtxt` and provides function-specific context
- Tracks return type coercion, divergence information, and parameter environments

- Contains the `span` where the variable was created and an optional `param_def_id`
