- `pkg/apis/core/validation/validation.go`:

  - `ValidateSecretUpdate()` - Prevents data changes and flag reversion for immutable Secrets
  - `ValidateImmutableField()` - Helper for immutable field validation

  - `pkg/registry/core/secret/strategy.go`

- `pkg/kubelet/secret/secret_manager.go` and `pkg/kubelet/configmap/configmap_manager.go` - Create watch-based managers with immutable detection callbacks
  
- `pkg/kubelet/util/manager/watch_based_manager_test.go` - Tests verifying immutable objects stop being watched

### 4. **Test Coverage**
- `test/e2e/common/storage/configmap_volume.go` - Conformance tests for ConfigMap immutability behavior
