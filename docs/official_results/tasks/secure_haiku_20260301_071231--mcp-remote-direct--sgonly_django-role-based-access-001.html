## Issue

- Input: `widget.render('field', '', attrs={'data-label': 'Value with "quotes"'})`

- Before fix: `data-label="Value with "quotes""` (broken HTML)

The `build_attrs` method in the `Widget` class (django/forms/widgets.py) was simply merging attribute dictionaries without escaping the values.
