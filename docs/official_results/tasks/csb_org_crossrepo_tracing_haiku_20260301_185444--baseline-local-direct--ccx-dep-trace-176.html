- Updated after successful processing via `api_state.request_.set_version_info()`

**6. Configuration Dispatch**
- **Listeners**: `LdsApiImpl::onConfigUpdate()` → `listener_manager_.addOrUpdateListener()`
- **Clusters**: `CdsApiImpl::onConfigUpdate()` → cluster manager configuration
