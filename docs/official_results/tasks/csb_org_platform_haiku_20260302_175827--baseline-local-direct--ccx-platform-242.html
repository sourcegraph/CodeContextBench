**The 9 Scheduler Framework Extension Points:**
1. **PreFilter** - Initial pod checks and node filtering
2. **Filter** - Filter nodes that cannot run the pod
3. **PostFilter** - Preemption plugin when pod can't be scheduled

6. **Reserve** - Update plugin state when scheduler cache is updated
7. **PreBind** - Last checks before binding a pod
