10. **CoordinatorRuntime.CoordinatorContext.append()** → Appends records to batch
11. **CoordinatorRuntime.CoordinatorContext.flushCurrentBatch()** → Flushes batch to log
12. **PartitionWriter.append()** → Writes to __consumer_offsets topic partition
