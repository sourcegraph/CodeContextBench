# NumPy dtype Compatibility Issue: Pandas Nullable Integers → Scikit-learn Preprocessing

## Executive Summary

When a pandas DataFrame with nullable integer types (e.g., `pd.Int64Dtype`) containing missing values is passed to scikit-learn's preprocessing functions (e.g., `StandardScaler`, `MinMaxScaler`), the conversion fails with a `ValueError: "cannot convert NA to integer"`.

The root cause lies in the pandas `BaseMaskedArray.astype()` method which explicitly prevents conversion of nullable integer arrays to integer dtypes when NA values are present.
