- **Location**: `compiler/rustc_hir_typeck/src/fn_ctxt/mod.rs` (line 44)

- **Key details**: Generic over lifetimes `'a, 'tcx`, stores parameter environment, return type coercion info, and divergence tracking

- **Role**: Defines how type variables are created and tracks their origins
- **Key details**: Stores span information and optional DefId for better diagnostics

The architecture follows a hierarchy where `InferCtxt` provides the low-level inference machinery in `rustc_infer`, while `FnCtxt` in `rustc_hir_typeck` builds on top to provide high-level function body type checking. `TypeVariableOrigin` is used throughout to track variable provenance for error reporting.
