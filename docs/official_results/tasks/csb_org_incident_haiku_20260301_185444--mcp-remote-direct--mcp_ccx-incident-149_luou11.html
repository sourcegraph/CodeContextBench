   - `ResolveToCSSPixels()` - Bridges to Rust evaluation via FFI
   - `DefaultPercentLengthToAppUnits()` - Applies overflow clamping

3. **Rust FFI Bridge** (`servo/ports/geckolib/glue.rs`):
   - `Servo_ResolveCalcLengthPercentage()` - Exported C function that calls Rust's resolve()

4. **Overflow Protection** (`gfx/src/nsCoord.h`):
   - `NSToCoordTruncClamped()` - Bounds-checks and clamps extreme values to safe integer range

### **Call Chain:**
C++ → `ResolveToCSSPixels()` → `Servo_ResolveCalcLengthPercentage()` → Rust `resolve()` → `resolve_internal()` → result clamping via `NSToCoordTruncClamped()`

The overflow handling works by allowing Rust's f32 arithmetic to naturally produce infinity values for operations like `1.0 / 0.0`, then safely clamping these at the C++ boundary before they're used in layout calculations.
