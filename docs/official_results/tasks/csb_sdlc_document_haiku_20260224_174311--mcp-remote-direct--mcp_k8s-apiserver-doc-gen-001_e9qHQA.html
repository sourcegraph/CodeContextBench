# Kubernetes API Server Library: Architecture and Extension Guide

## 1. Subsystem Overview

The Kubernetes API server library (`k8s.io/apiserver`) is a reusable framework for building Kubernetes-compatible REST APIs. It is the core component used by `kube-apiserver`, `kube-aggregator`, and `apiextensions-apiserver`. The library provides:

- **Request handling pipeline**: Authentication, authorization, admission control, and request dispatching
- **Storage abstraction**: Unified interface for persisting and retrieving objects via etcd
- **API group registration**: Mechanism for exposing resource types via REST endpoints
