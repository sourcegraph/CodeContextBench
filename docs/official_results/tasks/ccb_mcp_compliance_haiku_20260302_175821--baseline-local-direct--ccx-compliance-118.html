The **`RelatedFieldListFilter.has_output()` method (lines 248-253)** is where empty related-field filters are excluded:
```python
def has_output(self):
    if self.include_empty_choice:
        extra = 1
    else:
        extra = 0
    return len(self.lookup_choices) + extra > 1
```

This ensures filters with no choices (empty relations) are not rendered by requiring at least 2 options to display.
