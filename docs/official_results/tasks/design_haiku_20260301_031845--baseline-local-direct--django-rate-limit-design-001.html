   - Implements `process_request()` method to intercept requests before they reach views
   - Properly initializes with `get_response` parameter

2. **Rate Limiting Logic:**
   - Tracks requests per client IP using in-memory dictionary with timestamp lists
   - Configurable via Django settings:
     - `RATE_LIMIT_REQUESTS` (default: 100)
     - `RATE_LIMIT_WINDOW` (default: 3600 seconds / 1 hour)
   - Returns `HttpResponseForbidden` (403) when limit is exceeded with message including the client IP

3. **Request Tracking:**
   - Extracts client IP from `request.META.get('REMOTE_ADDR')`
   - Stores timestamps of requests for each IP
   - Filters out timestamps older than the configured window on each request
   - Automatically cleans up expired entries every 100 requests to prevent memory bloat

4. **Code Quality:**
   - Valid Python syntax (verified with py_compile)
   - Clear documentation and comments
   - Follows Django's code style and patterns
   - Properly scoped to `django/middleware/` directory
